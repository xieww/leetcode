# Excel 表列序号

给定一个 Excel 表格中的列名称，返回其相应的列序号。

例如，

```js
    A -> 1
    B -> 2
    C -> 3
    ...
    Z -> 26
    AA -> 27
    AB -> 28
    ...
```

示例 1:

```js
输入: "A";
输出: 1;
```

示例  2:

```js
输入: "AB";
输出: 28;
```

示例  3:

```js
输入: "ZY";
输出: 701;
```
